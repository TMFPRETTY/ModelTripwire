# Command Center Ranking

## Priority Order
When several things compete for attention, usually rank them like this:
1. blockers to today’s execution
2. approvals that are waiting on Jeremy
3. security/infrastructure risks
4. support issues with real customer impact
5. strong growth or product opportunities
6. routine status notes

Adjust only when a room-specific issue clearly matters more.

## Good Digest Pattern
A good digest is usually short enough to scan in under a minute.

Example shape:
- **Top priorities:** 2-4 bullets
- **Blockers / risks:** 1-3 bullets
- **Approvals waiting:** 0-3 bullets
- **Room highlights:** short grouped bullets
- **Recommended focus today:** 1-3 bullets

## Compression Rule
If a room has many minor updates, replace the list with one summary bullet unless an individual item changes a decision.
